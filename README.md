# Projeto-Tabuleiro-Next-React
Construindo um tabuleiro utilizando o react e o next.js
- O projeto consiste em criar um tabuleiro em uma página web, utilizando React e o Nextjs.
- Esse pequeno projeto é um desafio proposto pela escola de programadores Cod3r.

# O que foi utilizado 
- Utilizei o ReactJs e o NextJs nesse projeto
- Utilizei JavaScript 
- Utilizei Css3 e também códigos Jsx

# Para rodar o projeto 
- Para rodar o projeto é necessário ter o node e o React instalado em sua maquina
- Abra a pasta tabuleiro utilizando o terminal e execute o seguinte comando: npm rum dev
- Caso não tenha o react instalado em sua maquina execute o seguinte comando: npm i -g create-react-app
